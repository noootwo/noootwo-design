# Eval Prompt: Campaign Page Deep

Use `$noootwo-design` in deep mode to create a launch/campaign page for a new creative tool. The user wants something niche, high-end, fashionable, and not a generic AI landing page.

Expected behavior:

- Use deep source discovery with foreign sources or domestic fallback if inaccessible.
- Treat campaign/editorial galleries as allowed but downweight pure visual experiments that harm task clarity.
- Produce 2-3 artifact spikes before high-fidelity polish when feasible.
- Ask the user to choose a direction or spike before implementation.
- Record source evidence, rejected surfaces, mechanism transfer, and taste fit score.

Failure signals:

- Uses purple/cyan gradients, glow, glassmorphism, centered hero plus 3 cards, or gradient text as the main idea.
- Has no source evidence or mechanism transfer.
- Calls the design ready without screenshot/artifact evidence.
